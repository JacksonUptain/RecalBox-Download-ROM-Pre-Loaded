=======================================================================
# RECALBOX: Sistema Nintendo Entertainment System (Console da Tavolo)
=======================================================================

Copia qui le rom per Nintendo Entertainment System.
I file supportati sono: .unf .nes .zip .7z .unif

Questo sistema può essere emulato con 6 emulatori.
Tutti gli emulatori sono in questa lista, in ordine di compatibilità/semplicità di utilizzo:
libretro-nestopia supporta i file di .nes .unf .unif .zip .7z e NON è compatibile con Netplay.
libretro-fceumm supporta i file di .nes .unf .unif .zip .7z e NON è compatibile con Netplay.
libretro-fceunext supporta i file di .nes .unf .unif .zip .7z e NON è compatibile con Netplay.
libretro-quicknes supporta i file di .nes .unf .unif .zip .7z e NON è compatibile con Netplay.
libretro-mesen supporta i file di .nes .unf .unif .zip .7z e NON è compatibile con Netplay.
libretro-fbneo supporta i file di .zip .7z e  è compatibile con Netplay.

Requisiti di sistema:
- Il Controller è obbligatorio
- La Tastiera è non necessaria
- Il Mouse è non necessario

Questo sistema permette di utilizzare rom compresse in formato .zip o .7z.
Ma si tratta solamente di un file d'archivio. I file all'interno del .zip/.7z devono combaciare con le estensioni menzionate in precedenza.
Ogni .zip/.7z deve contenere solamente una rom compressa.
