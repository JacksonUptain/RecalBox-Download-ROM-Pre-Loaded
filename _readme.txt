=======================================================================
# RECALBOX: System Super Nintendo Entertainment System (Home console)
=======================================================================

Copy your Super Nintendo Entertainment System ROMs here.
Supported extensions are: .bs .sfc .fig .zip .gd3 .7z .swc .gd7 .dx2 .st .bsx .smc

This system can be emulated using 7 emulators.
These emulators are listed below, in order of compatibility/ease of use:
libretro-snes9x supports files of .smc .sfc .swc .fig .bs .st .7z .zip and is  Netplay compatible.
libretro-mesen_s supports files of .smc .sfc .swc .fig .bs .7z .zip and is  Netplay compatible.
libretro-snes9x2010 supports files of .smc .sfc .swc .fig .gd3 .gd7 .dx2 .bsx .7z .zip and is  Netplay compatible.
libretro-snes9x2005 supports files of .smc .sfc .swc .fig .gd3 .gd7 .dx2 .bsx .7z .zip and is  Netplay compatible.
libretro-snes9x2002 supports files of .smc .sfc .swc .fig .gd3 .gd7 .dx2 .bsx .7z .zip and is  Netplay compatible.
libretro-bsnes supports files of .smc .sfc .swc .fig .gd3 .gd7 .dx2 .bsx .7z .zip and is  Netplay compatible.
libretro-bsneshd supports files of .smc .sfc .swc .fig .gd3 .gd7 .dx2 .bsx .7z .zip and is  Netplay compatible.

Device requirements:
- A controller is required
- A keyboard is unusable
- A mouse is optional

This system allows to use compressed roms in .zip and or .7z.
But it is only an archive file. Files inside the .zip/.7z must match with extensions mentioned above.
Each .zip/.7z file must contain only one compressed rom.
